# ISDRC

## Innovation and Skill Development Research Centre

ISDRC is the official repository for the Innovation and Skill Development Research Centre.  
This repository hosts the source code and resources for the ISDRC website, including member information, project details, event updates, and downloadable materials.

## Repository Structure

- `index.html` – Main landing page  
- `members.html` – Member profiles page  
- `projects.html` – Projects showcase  
- `gallery/` – Event and activity images  
- `members.json` – Member data (JSON-based)  
- `projects.json` – Project data (JSON-based)  
- `script.js` – JavaScript for dynamic content rendering  
- `styles.css` – Website styling  
- `server.js` – Optional local development server  

## Features

- Static website hosted using GitHub Pages  
- Dynamic content rendered from JSON files  
- Member profiles with images  
- Project listings and descriptions  
- Event and activity gallery  
- Easy content updates without modifying HTML  

## Technologies Used

- HTML5  
- CSS3  
- JavaScript  
- JSON  
- Node.js (optional, for local testing)
